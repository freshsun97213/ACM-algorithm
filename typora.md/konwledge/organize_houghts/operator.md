#### 赋值语句及其赋值运算符

“=”就是赋值运算符，在C和C++ 里面，是可以出现外观奇怪 的代码，比如：

`````c++
int a,b,c;
a = b = c = 0;//类似这样的代码是可以出现的
`````

!!! 注意的是：赋值运算符的顺序是从右向左赋值的，和其他的运算符不同;
